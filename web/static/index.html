STRAT i
Extraction strategy in Unstructured: auto lets the library pick, fast is lighter, hi_res uses high‑resolution vision for layout accuracy.
 -
INF‑TBL i
Infer table structure (rows and cells) from detected lines/glyphs. When off, tables are generic blocks.
 -
CHUNK i
How elements are grouped: by_title uses headings as section boundaries; basic creates size‑based text chunks.
 -
LANG i
Primary document language. Arabic toggles RTL previews and prioritizes Arabic OCR.
 -
TOK i
Approximate target tokens per chunk (LLM). If provided, max_characters ≈ tokens × 4.
 -
CHARS i
Hard character cap per chunk. Used if max_tokens is not set.
 -
NEW i
Start a new chunk after this many characters have been appended.
 -
COMB i
If a chunk is smaller than this, merge it with the next. Applies to by_title.
 -
OVRLP i
Trailing characters to overlap between adjacent chunks to preserve context.
 -
INCL i
Include original atomic elements alongside chunk spans in outputs/overlays.
 -
OVRLP* i
Apply overlap to all boundaries, including section breaks. When off, overlap only within contiguous text.
 -
MPAGE i
Allow by_title chunks to span multiple pages until the next title appears.
 -
PDF i
Input PDF used to generate this run.
 -
PAGES i
Pages processed (list or ranges, e.g., 5-7,12).
 -
TAG i
Optional variant tag appended to the run slug for comparison.

Avg Coverage i
Macro average of per-table coverage (recall): share of gold rows covered by the selected chunks.
-
Avg Cohesion i
Macro average of per-table cohesion = 1 / selected_chunk_count. Maximized when a table stays in a single chunk.
-
Avg F1 i
Macro average of per-table F1 where F1 = harmonic mean of coverage and cohesion.
-
Micro Coverage i
Coverage weighted by gold-row counts across tables (larger tables contribute more).
